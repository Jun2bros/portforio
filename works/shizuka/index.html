Shizuka Official Website

Type：Website Launch Date：2022. 08 Url： https://shizukatou22.com/

junhyeongyu junhyeongyu junhyeongyu junhyeongyu

フリーランスのイラストレーター、および彫刻家として活動されている「しずか」さまの公式ウェブサイトを制作させていただきました。クライアントさまのメインコンセプトである魔法少女の世界観を存分に感じていただきながら、サイト自体も作品の一部として楽しめる、魅せられるようなサイトを目指しました。

ターゲット
お仕事依頼をされる企業さまや個人の方、クライアントさまのファンの方々
目的
実績や作品、SNSリンクなどを集積し、今後の活動において名刺代わりとして使用していただくため
情報設計
クライアントさまのこれまでの活動を最大限に伝えられるよう、作品中心のサイト構成を心がけました。特にイラスト作品は仕事依頼での実績と創作作品のページを分け、見せ方も明確に変えることで、経歴はきちんとアピールしつつもをワクワクしたユーザー体験を両立させることを意識しました。また、彫刻活動もされているということをアピールするために彫刻作品のページを設け、彫刻家としての一面も周知できるようにしました。
デザイン
ファーストビューで一気に世界観に惹き込めるよう、トップにはキャラを動かした動画を埋め込みました。背景色にはノイズやグラデーションを多用し、幻想的な雰囲気を表現。魔法陣のあしらいや魔導書モチーフのメニュー、流れるステッキなど、随所に魔法少女の要素を散りばめ、世界観を統一しました。また彫刻のページでは2次元から3次元へのギャップを見せるため、作品に影をつけ、実際にサイトの中に置いてあるようなデザインに仕上げました。
制作範囲・期間
企画、ワイヤーフレーム：1週間
デザイン：2週間
コーディング：1ヶ月
使用ツール
Illustrator / Photoshop / Premiere Pro / XD / Visual Studio Code
