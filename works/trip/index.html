Trip Media Site Banner

Type：Banner
Creation Date：2022. 08

Trip Media Site Banner - Trip Media Site Banner - Trip Media Site Banner - Trip Media Site Banner -

旅行メディアサイトに設置されたコラム記事のアイキャッチをテーマに、架空のサイト内バナーを制作しました。和風のモチーフをふんだんに使用しベースは和テイストにしながらも、英字やメリハリのある配色を使い、和モダンな雰囲気に仕上げました。またサイズ展開として、サイト内用と広告用に2サイズ制作しました。

サイズ
980 × 450px（長方形） / 1080 × 1080px（正方形）
ターゲット
京都旅行を控えており、下調べのためメディアサイトを訪れた20〜40代の男女
目的
嵐山での散策ルートや寄り道できるスポットを紹介し、実際に訪れてもらうため
レイアウト
縦書きのデザインのため、視線が右上からN字型に流れることを想定し、テキストエリアを右上に大きく配置しました。また正方形の方では要素が三角形の構図になるように配置することでレイアウトに安定感を持たせ、重心でピタッと止まって見えるよう意識しました。
デザイン
和紙のテクスチャや松の葉をイメージしたトリミング、市松模様や霞のあしらいなど、日本伝統的なモチーフを多用しながらも、丸みを帯びたシルエットや英字を取り入れることで、現代的な和モダンデザインを目指しました。テキストエリアでは「嵐山」の文字のみフォントやカラーを変え、若干の作字も加えて目立つように工夫しました。また配色はターゲット層が20代以上の大人であることから、落ち着いたシックなカラーを選択し、深めの朱色でアクセントを加えています。
制作時間
5時間（2枚合計）
使用ツール
Illustrator / Photoshop
